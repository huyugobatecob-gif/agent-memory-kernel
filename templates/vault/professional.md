# Professional Memory

Use this lane for project rules, decisions, successful patterns, failed
attempts, operational lessons, and working context.

Default behavior:

- retrieve as the default work lane when policy allows it;
- keep assistant, tool, web, and external-document claims reviewable;
- filter by scope, namespace, trust, sensitivity, lifecycle state, and prompt
  budget;
- preserve provenance, review history, review queues, and lifecycle state on
  export/import.
